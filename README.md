### **애플리케이션 설명**
- CoreData를 활용해 포토카드 형태로 다이어리 작성이 가능한 앱입니다.
- 사용자 위치 정보를 기반으로 메인 포토카드에 날씨 아이콘이 추가 됩니다.
- 추억 및 최신 각 셀 Long PressGeture 시 수정 및 삭제가 가능합니다.
- 원하는 데이터는 캘린더 뷰를 이용해 일자별로 확인 가능합니다.
- 원하는 데이터는 본문 내용 텍스트 기반으로 검색 가능합니다.
